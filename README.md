# 📱 VTcampro – Web Camera App

VTcampro is a powerful **web-based camera application** built using **HTML, CSS, and JavaScript**.  
It simulates a modern mobile camera with multiple modes, AI features, and a clean mobile UI.

---

## 🚀 Features

### 📸 Camera Modes
- Photo Mode
- Video Recording
- Portrait Mode (blur effect)
- Night Mode (low-light simulation)

### 🤖 AI Features
- Face Detection (TensorFlow.js)
- Smart Portrait Blur

### 🎨 Filters
- Grayscale
- Sepia
- Brightness
- Contrast

### 🎥 Video Features
- Record video with audio
- Save recordings locally

### 📱 UI/UX
- Mobile-style camera layout
- Smooth controls
- Zoom support
- Camera switch (front/back)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Web APIs:
  - getUserMedia
  - MediaRecorder
  - Canvas API
- TensorFlow.js (for AI)
